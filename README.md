"I am the standard. They have to beat me; I will not beat myself."

## **Overview**

This application is the digital enforcement of **The Mendelsohn Protocol**, a strategic mental framework designed for the **Loriella Challenge**. It replaces "feeling" with **data** and "deliberation" with **execution**.

The app serves two functions:

1. **The Caddie:** Displays the specific, pre-determined game plan for every hole (Shorts & Longs).  
2. **The Analyst:** Logs actual performance data (Disc selection, Shot shape, Rating) to refine the bag for Tournament Day.

## **Core Axioms**

* **Boring Golf:** A stroke saved by safety is worth more than a stroke gained by brilliance.  
* **Tee Box Discipline:** All thinking happens *behind* the box. Inside the box is for execution only.  
* **The 10-Second Rule:** Emotional processing is allowed for exactly 10 seconds. Then, it is evicted.

## **Setup & Installation**

### **1\. Clone the Repository**

git clone \[https://github.com/Lukemendels/mks-tracker.git\](https://github.com/Lukemendels/mks-tracker.git)  
cd mks-tracker

### **2\. Install Dependencies**

python3 \-m venv venv  
source venv/bin/activate  
pip install \-r requirements.txt

### **3\. Configure Secrets**

Create a .streamlit/secrets.toml file (**Do not commit this\!**):

SUPABASE\_URL \= "your\_project\_url"  
SUPABASE\_KEY \= "your\_public\_anon\_key"

### **4\. Run the App**

streamlit run tracker.py

## **Database Schema (Supabase)**

Run this SQL in your Supabase Editor to create the table and enable Row Level Security (RLS):

create table practice\_notes (  
  id bigint generated by default as identity primary key,  
  created\_at timestamp with time zone default timezone('utc'::text, now()) not null,  
  hole\_number bigint,  
  layout text,  
  disc\_used text,  
  result\_rating bigint,  
  notes text  
);

alter table practice\_notes enable row level security;

\-- Only allows authenticated users to access data  
create policy "Allow authenticated access"  
on practice\_notes  
for all  
to authenticated  
using (auth.role() \= 'authenticated')  
with check (auth.role() \= 'authenticated');
